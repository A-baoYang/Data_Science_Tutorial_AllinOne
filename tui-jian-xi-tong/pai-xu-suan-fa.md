# 排序算法

### 常用算法

* 線性模型
  * LR
  * FM
* 樹模型
  * Decision Tree / Random Forest
  * GBDT / XGBoost / LightGBM
* 單目標神經網絡
  * DNN
  * DeepFM
  * DCN
* 多目標學習
  * ESMM
  * MMoE

### 排序流程

* 粗排：將召回候選集的規模排序篩選到萬以內等級
* 精排：再從萬等級做個性化排序
* 重排：最後經更一些業務規則（如：去已讀、去重複、打散、多樣性、固定宣傳所需）等，目的是為了符合業務需求及提升用戶體會

### 模型訓練流程

1. 建構訓練樣本
   1. 正面樣本：有曝光有點擊
   2. 負面樣本：有曝光無點擊
2. 樣本預處理
   1. 排除異常樣本
   2. 特徵工程
3. 根據業務選擇合適的模型
4. 模型訓練調參
